COLLAUDO BANCHI OPTOFLASH

Procedura per il setup ed il collaudo metrologico

 Video

  1. Avviare il software di gestione del driver tramite l'icona sul desktop.
  2. Creare un "gold drive".
  3. Creare una connessione "Direct Access USB".
  4. Selezionare la porta seriale dell'azionamento (in genere quella con il numero più alto) e attivare la connessione.
  5. Per caricare i parametri dell'azionamento, selezionare il tab "Parameters" e successivamente "Textual" in alto a sinistra nella schermata.
  6. Nella finestra di dialogo che appare:
    1. Selezionare "Set from file". (IMPORTANTE!)
    2. Caricare il file contenente la configurazione del driver (Parametrs_SRP5080.gprm).
    3. Premere il tasto "Download".
  7. Al tremine della procedura premere "Close".

Ora è necessario verificare che i parametri siano stati coricati correttamente.

  1. Selezionare "Drive Setup and Motion".
  2. Abilitare l'azionamento.
  3. Effettuare una rotazione del motore.
  4. Disabilitare l'azionamento.
  5. Verificare che i parametri siano stati caricati correttamente nell'azionamento, digitando il comando "[AA10]" nella finestra apposita.
    L'indirizzo IP dell'azionamento deve essere 169.254.2.2
  6. Digitare il comando "SV" nella finestra apposita per salvare i parametri nella memoria del'azionamento.
  7. Disconnettere l'azionamento e chiudere il software di gestione.
    Non è necessario salvare l'ambiente di lavoro.

La configurazione dell'azionamento è terminata.

 Video

  1. Avviare il software di installazione dell'interfaccia OPTIC.
  2. Selezionare Inglese come lingua di installazione.
  3. Seguire la procedura guidata ed inserire:
    1. nel campo Username il valore Optoflash
    2. nel campo Organization il valore Marposs S.p.A.
  4. NON abilitare l'avvio automatico dell'interfaccia OPTIC.
  5. Terminare l'installazione del software.

Ora è necessario abilitare l'integrazione dell'interfaccia OPTIC.

  1. Dal menù START di Windows cercare la voce "Optic - Tools".
  2. Dalla finestra che si apre, lanciare QspcwAndOptic.exe
  3. Nel popup che appare cliccare su INTEGRATION Optic on startup.

L'installazione è terminata.

  1. Creazione di una configurazione OPTO di base
    1. Avviare il software di configurazione OPTO utilizzando il tool oq_config.exe
      che si trova nella cartella C:\Program Files (x86)\QSPCW\Bin\Optoflex\.
    2. Dopo una serie di schermate DOS apparirà una finestra tramite la quale si procederà alla definizione del tipo di banco e di forchetta.
      1. Dal primo menu a tendina scegliere il tipo di banco (nell'esempio un OF motorizzato Heidenhain).
      2. Dal secondo menu a tendina scegliere il tipo di forchetta(nell'esempio una forchetta 3 sonde).
      3. Abilitare l'integrazione con l'interfaccia OPTIC e premere il tasto Execute.
    3. Quando il tool ha completato la copia dei file della configurazione di base, premere il tasto Exit.
      In questo modo è stata creata la cartella C:\ProgramData\Marposs\QSPCW\Common\OptoWork\.
