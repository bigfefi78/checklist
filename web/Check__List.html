PROCEDURA DI COLLAUDO BANCHI OPTOFLASH


Di seguito la procedura per la configurazione ed il collaudo di un banco Optoflash

  1. Lanciare il software di gestione del driver.
  2. Aggiungere un gold drive.
  3. Eseguire la connessione al driver utilizzando l'opzione Direct Access USB.
  4. Caricare nella memoria del driver il file di configurazione, prestando attenzione a selezionare l'opzione set from file.
